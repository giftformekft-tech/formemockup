Price Visibility Debug

Instrukciók:

  1. Nyisd meg az érintett termékoldalt
  2. Nyisd meg a DevTools-t (F12)
  3. Menj a Console tabra
  4. Másold be ezt a kódot és nyomd meg Enter-t:
// Ellenőrizd az HTML osztályokat
console.log('HTML classes:', document.documentElement.className);

// Ellenőrizd hogy létezik-e a .price elem
const priceEl = document.querySelector('.summary > p.price');
console.log('Price element:', priceEl);

// Ha létezik, ellenőrizd a stílusait
if (priceEl) {
    const styles = window.getComputedStyle(priceEl);
    console.log('Price opacity:', styles.opacity);
    console.log('Price visibility:', styles.visibility);
    console.log('Price display:', styles.display);
}

// Ellenőrizd hogy a MG_VARIANT_DISPLAY betöltődött-e
console.log('MG_VARIANT_DISPLAY:', typeof MG_VARIANT_DISPLAY);

// Kényszerítsük láthatóvá (teszt)
if (priceEl) {
    priceEl.style.opacity = '1';
    priceEl.style.visibility = 'visible';
    priceEl.style.display = 'block';
    console.log('✅ Kényszerített láthatóság alkalmazva');
}

Küldd el nekem a console kimenetét!

Gyors megoldás (ideiglenes):

Ha a fenti kód után MEGJELENIK az ár, akkor a probléma a CSS specificitás. Ebben az esetben egyszerűen töröljük az ár elrejtését TELJESEN.
